## 1. 设计原则

### 1.1 核心理念

**对话即工作流。** 所有操作都应该发生在对话线程中，用户不需要在多个页面间跳转来完成一个创作任务。AI 的思考过程、工具调用、结果输出，都应该在同一个时间线里自然展开。

### 1.2 五条铁律

1. **统一性**：整个应用是一个连续的对话体验，不是四个独立页面的拼凑
2. **透明性**：AI 在做什么、用了什么工具、花了多长时间，用户全程可见
3. **层次感**：信息有主次，核心内容突出，辅助信息收敛
4. **呼吸感**：留白是设计语言，不是浪费空间
5. **确定性**：每个交互都有明确反馈，用户永远知道发生了什么

## 2. 全局布局

### 2.1 当前问题

- 侧边栏只有 4 个图标，无文字标签，认知成本高
- 每个页面有独立的 header，风格不统一
- ChatView 和 DataView 是两个完全不同的应用，导航割裂
- 没有全局搜索
- 没有统一的上下文状态（当前在做什么、有什么任务在跑）

### 2.2 目标布局

采用 Codex 风格的 **三栏布局**，但在知更的场景下做适配：

```

│                    macOS 标题栏（隐藏）                     │
├────────┬─────────────────────────────────────────────────┤
│        │                                                 │
│  侧边栏  │                   主内容区                       │
│  240px  │                                                 │
│        │  ┌─────────────────────────────────────────┐    │
│        │  │                                         │    │
│  ┌──┐  │  │         根据当前页面渲染                    │    │
│  │LOGO│  │                                         │    │
│  └──┘  │  │  创作台：对话流 + 右侧物料面板             │    │
│        │  │  数据概览：数据面板                        │    │
│  导航   │  │  灵感库：选题面板                         │    │
│  ───── │  │  设置：配置表单                          │    │
│  创作台  │  │                                         │    │
```

### 2.3 侧边栏设计

**宽度**：240px（可折叠到 56px）

**结构**：

```
│                      │
│  ──────────────────  │  ← 分割线
```

**交互细节**：

- 导航项 hover 时背景变为 `surface-container`，文字变为 `on-surface`
- 当前激活项：左侧 3px `primary` 色竖条 + `primary` 色文字 + `surface-container` 背景
- 对话历史项：最多显示 2 行截断，hover 时显示完整标题
- 对话历史项右侧显示时间戳（相对时间："2小时前"、"昨天"）
- 对话历史项 hover 时右侧出现删除按钮
- 侧边栏底部可以折叠/展开（点击 logo 区域或快捷键）

## 3. 创作台（ChatView）重新设计

### 3.1 当前问题

- 消息气泡样式单一，用户和助手消息区分不够明显
- MaterialCards 嵌套在气泡里，tab 布局在窄空间内很拥挤
- 工具调用状态只是一行文字（"正在分析..."），没有视觉层次
- 流式输出的打字动画不够自然
- 没有消息操作栏（复制/重试/编辑）
- 欢迎页太简单，没有引导性

### 3.2 目标体验（对标 Codex）

#### 3.2.1 对话流

**消息布局**：全宽消息（非气泡），助手消息左对齐，用户消息右对齐但不使用气泡背景。

#### 3.2.2 工具调用卡片

**样式**：圆角卡片，左侧带 2px 色条（不同工具不同颜色），可折叠展开。

```
┌─ 🔧 分析脚本 ────────────────────── 0.8s ✓ ─┐
│                                              │
│  ▸ 展开详情                                   │  ← 默认折叠，点击展开
│                                              │
└──────────────────────────────────────────────┘
```

**展开状态**：

```
┌─ 🔧 分析脚本 ────────────────────── 0.8s ✓ ─┐
│                                              │
│  ▾ 收起                                      │
│                                              │

│                                              │
│  输出                                         │
│  ┌────────────────────────────────────────┐  │
│  │ {                                       │  │
│  │   "videoType": "科技测评",              │  │
│  │   "topic": "华为Mate70 Pro深度体验",    │  │
```

**状态颜色**：

| 状态 | 色条颜色 | 图标 |
|---|---|---|

| 失败 | `red-400` | `error` |
| 等待中 | `outline-variant`（灰色） | `schedule` |

#### 3.2.3 物料展示面板

**核心改动**：物料从"嵌套在气泡内的 Tab 卡片"改为"独立的全宽面板"，视觉层次更高。

**Tab 栏设计**：

```
[B站]  [YouTube]  [抖音]  [小红书]  [拍摄清单]
```

- 选中态：底部 2px `primary` 色线条 + 文字变 `primary`
- 未选中态：文字 `on-surface-variant`，hover 时变 `on-surface`
- Tab 栏可以横向滚动（小屏幕时）

**物料内容区**：

- 点击"复制全部" → 复制当前平台所有字段，toast 提示"已复制到剪贴板"
- 复制格式：纯文本（标题 + 换行 + 简介 + 换行 + 标签）

#### 3.2.5 消息操作栏

每条消息 hover 时底部出现操作栏：

```
┌─────────────────────────────────────────────┐
│  助手消息内容...                              │
```

## 4. 数据分析页（DataView）重新设计

### 4.1 当前问题

- 顶部操作栏太拥挤（模式切换 + 平台选择 + 输入框 + 3个按钮全部挤在一行）
- 左侧边栏和 ChatView 的侧边栏风格不一致
- 表格在小屏幕上体验差

## 7. 全局交互规范

### 7.1 流式输出动画

**打字效果**：逐字显示，速度约 30-50 字/秒（可配置）。

**工具调用动画**：

1. 工具调用开始：卡片从左侧滑入 + 色条脉冲动画
2. 执行中：图标旋转（`sync` 图标 CSS animation）
3. 完成：图标变为 `check_circle` + 色条停止脉冲
4. 失败：图标变为 `error` + 色条变红色

**流式文本光标**：

```
已为你生成三平台发布物料█  ← 闪烁的光标（绿色方块）
```

### 7.2 过渡动画

| 场景 | 动画 | 时长 |
|---|---|---|
| 页面切换 | 淡入 + 轻微上移 | 200ms ease-out |
| 侧边栏折叠/展开 | 宽度过渡 | 250ms ease-in-out |
| 工具调用卡片展开/折叠 | 高度过渡 | 200ms ease-out |
| 消息出现 | 淡入 + 上移 8px | 150ms ease-out |
| 按钮 hover | 背景色过渡 | 100ms |
| Tab 切换 | 内容淡入 | 150ms |

### 7.3 快捷键

| 快捷键 | 功能 |
|---|---|
| `Cmd/Ctrl + N` | 新建对话 |
| `Cmd/Ctrl + K` | 全局搜索（搜索对话/内容） |
| `Cmd/Ctrl + ,` | 打开设置 |
| `Cmd/Ctrl + Enter` | 发送消息 |
| `Cmd/Ctrl + Shift + C` | 复制最后一条助手消息 |
| `Escape` | 停止生成 / 关闭面板 |
| `Cmd/Ctrl + [` | 折叠侧边栏 |
